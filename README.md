# ⚽ Football Team Manager (C++)



This project is a simple football team management simulation written in **C**++ programming language.

It models players, teams, transfers, and match simulation using object-oriented programming principles.



The project was developed as part of my university coursework for the subject **Programmierung 1**.



---



\## 📚 Project Overview



The application demonstrates core OOP concepts such as:



\* Class design and encapsulation

\* Operator overloading

\* Exception handling

\* STL containers and algorithms

\* Modular project structure (header / implementation separation)



It simulates real football management features including:



\* Player creation with positions and transfer fees

\* Player statistics (goals, assists, cards)

\* Team roster management

\* Signing and releasing players

\* Player transfers between teams

\* Automatic starting lineup generation

\* Match simulation with random goals and assists



---



\## 🧩 Project Structure





FootballTeamManager:



* main.cpp         # Program entry point
* player.h / .cpp  # Player class and statistics logic
* team.h   / .cpp  # Team management and transfers
* match.h  / .cpp  # Match simulation engine



---



\## 🧑‍💻 Features



\### Player System



\* Name, position, and transfer fee validation

\* Performance statistics tracking

\* Dynamic rating calculation



\### Team Management



\* Squad handling

\* Duplicate player prevention

\* Signing and releasing players

\* Transfer operations between teams

\* Formation / lineup generation



\### Match Simulation



\* Random score generation

\* Goal and assist assignment

\* Automatic stat updates

\* Final match result reporting
